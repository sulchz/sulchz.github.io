Software

I'd been interested in tech and software for a while, and had some fun in school doing stuff with autohotkey (Windows only). I got into linux late 2018, on Manjaro, largely due to wanting to rice. At some point I'll be adding pictures of stuff I made, some of which I'm quite proud of :p. I started off on i3wm, which I still reccomend to people new to tiling window managers simply due to how popular it is and the on-the-fly recompiling, it's a lot simpler to use than stuff like DWM which is what I'm on now. Again, at some point I need to point to the current configs which I use. Feel free to use them/edit them if you'd like.

Anyways, after Manjaro I spent a lot of time bouncing around various distros; I had several spare PCs at the time and no shortage of time. My reccomendations now are (in no particular order):

  • Arch Linux - has somewhat of a negative repuation within the community for being elitist, and from experience it's one that's well deserved. Still, if you're willing to tinker around with stuff it's great simply due to how good the wiki is. Also, the AUR is amazing, probably the only thing keeping me on it at this point.
  • Void Linux - Another really nice bleeding-edge distro, with a very nice package manager. There were some schenanigans regarding the bus-factor with it a while back but it seems to have been resolved. The ISOs are a bit outdated as of today (14-01-21) but the repos are fresh. Also, the community is somewhat more welcoming for what is in effect a very similar distro.
  • Debian (netinst) - My go-to for a minimal, easy to use install on a system that I don't/can't run bleeding edge. I don't really see the point of running some downstream variant, particularly when you can build as light of a system from scratch.
  • st - the suckless (or simple) terminal. Patches I use for it are scrollback, alpha (alongside compton) and clipboard. Might add stuff later if i see the need to.
